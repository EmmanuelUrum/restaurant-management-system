# restaurant-management-system

## Description
This is a database-driven, Web-based restaurant management system application

## Basic Features
- Customer management
- Menu management
- Order processing
- Staff management
- Payment management

## Advanced Features
- Recommendation system
- Sales analytics dashbaord

## Softwares Used
- Python
- Flask
- MySQL
- JavaScript
- HTML

 ## How to Run
 1. Start MySQL and create the database
 2. Run backend:
    python restaurantdatabase.py
3. Open frontend:
   frontend.html
